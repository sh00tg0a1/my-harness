# Product Sense — fancy-text

## Target users

- **Content creators** — want eye-catching text for social media
- **Developers** — need quick visual demos or placeholder effects
- **Casual users** — just want to play with cool text effects

## Value proposition

Type any text in any language → instantly see it rendered with beautiful effects. No sign-up, no config, no learning curve.

## Key metrics (future)

| Metric | Target |
|--------|--------|
| Time to first "wow" | < 3 seconds |
| Styles available | ≥ 5 |
| Language support | Full Unicode |
