# Core Beliefs — fancy-text

1. **Text is universal** — every human language should render beautifully.
2. **Effects should feel magical** — smooth animations, surprising transitions, attention to timing.
3. **Zero friction** — no login, no config, no tutorial. Type and see.
4. **Extensible styles** — adding a new style should be a 3-file change (type, picker entry, CSS class).
5. **Performance matters** — even with fancy effects, the UI must stay responsive under 60fps.
